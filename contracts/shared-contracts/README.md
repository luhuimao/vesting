## Shared Contracts

Smart contracts to be shared across Sablier projects and packages.

## Usage

Install the module:

```bash
$ yarn add @sablier/shared-contracts
```

And import it in your solidity project:

```solidity
import "@sablier/shared-contracts/mocks/ERC20Mock.sol";
```
